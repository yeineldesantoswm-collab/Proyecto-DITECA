Hola! Dentro de este repositorio encontramos todas las documentaciones dentro del proyecto DITECA.
Este es un proyecto usado dentro de la materia Conmutación y Enrutamiento del Instituto Tecnológico de las Américas (ITLA) 
Este proyecto se basa en la creación de una empresa ficticia contando con configuraciones de entornos reales dentro de un entorno virtualizado de PNETLAB
Podemos encontrar Switches, Routers, Servidores y VPN dentro de estas configuraciones, todo detallado dentro de las carpetas de este proyecto.
